# Jobify Portal

A modern, premium SaaS web application that connects talented professionals with world-class companies. 

## 🚀 Tech Stack

- **Framework:** React + Vite
- **Styling:** TailwindCSS v4
- **Routing:** React Router DOM
- **Icons:** Lucide React
- **Fonts:** Google Fonts (Inter & Outfit)

## ✨ Features

- **Modern Glassmorphism UI:** Features a sleek pill-shaped sticky navbar with frosted glass gaussian blur that activates on scroll.
- **Dynamic CSS Grid Layouts:** Responsive sections highlighting trusted companies, featured jobs, features, and testimonials.
- **Premium Aesthetics:** Micro-animations alongside carefully curated hover effects that create a responsive, engaging user experience.
- **Clean Architecture:** Component-driven development organized into modular UI sections.

## 🛠️ Components

- **Navbar:** Sticky pill navigation with smooth scroll detection.
- **Hero:** Vibrant dual-column layout boasting quick action search features.
- **Companies:** Interactive logo wall that ignites with color on hover.
- **FeaturedJobs:** Engaging job listing cards utilizing custom soft shadows.
- **HowItWorks & Features:** Highlight platforms mechanics utilizing clear typography and iconography.
- **Testimonials:** Verified user story cards designed to inspire trust.
- **Footer & CTA:** Comprehensive routing footer designed to seal user engagement.

## 💻 Getting Started

### Prerequisites
Make sure you have Node.js installed.

### Installation

1. Clone the repository and navigate to the project directory:
   ```bash
   cd Amdox-Project1
   ```

2. Install all dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open `http://localhost:5173` in your browser to view the app!
